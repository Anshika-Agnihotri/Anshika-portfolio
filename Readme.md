# Personal Portfolio Website

This is a responsive personal portfolio website created using HTML, CSS, Bootstrap and JavaScript.

The website includes different sections such as:
- Home
- About
- Skills
- Projects
- Contact

## Features

- Responsive design for mobile and desktop
- Modern dark UI
- Animated profile section
- Project cards with hover effects
- Contact form layout
- Social media links
- Bootstrap navbar and grid system

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Font Awesome

## Purpose of the Project

This project was created as part of a Full Stack Development Internship task to practice:
- Responsive web design
- Advanced CSS styling
- Layout structuring
- Bootstrap components
- Frontend development skills
